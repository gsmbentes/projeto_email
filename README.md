# Sistema de Gerenciamento de Contas e Autenticação (Python)

Este projeto simula um sistema de criação de e-mails e autenticação em sites, desenvolvido com foco em **Programação Orientada a Objetos (POO)** e **Segurança de Dados**. O objetivo principal é demonstrar a aplicação de regras de negócio estritas para validação de senhas e domínios.

## 🚀 Funcionalidades

- **Criação de Conta:** Validação de e-mails @gmail.com e exigência de senhas fortes (letras, números e caracteres especiais).
- **Segurança:** Uso de atributos privados e protegidos (Encapsulamento).
- **Autenticação:** Sistema de login para acesso a sites simulados.
- **Validação de Domínio:** Verificação de URLs (ex: deve começar com `www.` e terminar com `.com.br`).
- **Gestão de Senha:** Funcionalidade de redefinição de senha com verificação de integridade.
- **Interface via Terminal:** Menu interativo para navegação do usuário.

## 🛠️ Tecnologias Utilizadas

- **Python 3.13.2**
- **Conceitos de POO:** Classes, Métodos, Encapsulamento, Atributos Privados.
- **Tratamento de Exceções:** Uso de `raise`, `ValueError` e `PermissionError` para fluxos de erro robustos.
- **Lógica de Strings:** Manipulação e validação avançada de textos (Strings).

## 📂 Estrutura do Projeto
      
- `parte1.py`: Contém a classe `Email` com toda a lógica de negócio e validações.
- `main.py`: Script de execução que gerencia a interface com o usuário e o fluxo de dados.
